# JS第三天

  • 构造函数会导致内存浪费问题

1. 面向过程:

  1. 面向过程: 分析解决问题所需的步骤 用函数一步步实现 用时使用依次调用
  2. 按照分析好的步骤 按步骤解决问题

2. 面向对象:

  1. 面向对象: 把事务分解成对象 由对象之间分工合作
  2. 是以对象功能划分问题 而不是步骤
  3. 面向对象的特征: 封装、继承、多态

3. 面向过程和面向对象的区别:

  1. 面向过程性能高 但没有面向对象易维护、复用、扩展
  2. 面向对象性能低

4. 复习:

  1. JS面向对象需借助构造函数来实现
  2. 构造函数存在浪费内存问题
